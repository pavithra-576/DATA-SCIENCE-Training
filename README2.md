# Power BI Training

A structured, hands-on training program for learning Microsoft Power BI — from data import and modeling to DAX, visualization, and publishing dashboards.

## 📌 Overview

This training is designed to take learners from beginner to job-ready in Power BI. It covers the full workflow: connecting to data sources, transforming and modeling data, writing DAX measures, building interactive reports, and deploying to the Power BI Service.

## 🎯 Who This Is For

- Beginners with no prior Power BI experience
- Excel users looking to move into self-service BI
- Data analysts wanting to strengthen data modeling and DAX skills
- Anyone preparing for the PL-300 (Power BI Data Analyst) certification

## ✅ Prerequisites

- A Windows machine (Power BI Desktop is Windows-only)
- Basic understanding of spreadsheets (Excel or similar)
- No prior programming experience required
- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed (free)
- A free [Power BI Service](https://app.powerbi.com/) account (optional, for publishing modules)

## 🗂️ Course Structure

| Module | Topic | Description |
|--------|-------|-------------|
| 01 | Introduction to Power BI | Power BI ecosystem, Desktop vs Service, interface tour |
| 02 | Connecting to Data | Importing from Excel, CSV, SQL, Web, and other sources |
| 03 | Power Query & Data Transformation | Cleaning, shaping, merging, and appending data |
| 04 | Data Modeling | Relationships, star schema, cardinality, cross-filter direction |
| 05 | DAX Fundamentals | Calculated columns, measures, basic functions |
| 06 | Intermediate DAX | Time intelligence, CALCULATE, filter context |
| 07 | Data Visualization | Charts, slicers, filters, custom visuals |
| 08 | Reports & Dashboards | Building interactive, multi-page reports |
| 09 | Publishing & Sharing | Power BI Service, workspaces, apps, sharing reports |
| 10 | Row-Level Security (RLS) | Securing data at the user level |
| 11 | Performance & Best Practices | Optimization, naming conventions, folder structure |
| 12 | Capstone Project | End-to-end dashboard build using a real-world dataset |



## 🚀 Getting Started

1. Clone or download this repository.
2. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
3. Open the `modules/` folder and start with `01-introduction`.
4. Follow along with the exercise files in each module folder.
5. Compare your work with the corresponding file in `solutions/` if you get stuck.

## 🛠️ Tools Used

- Power BI Desktop
- Power BI Service (Cloud)
- Power Query (M language)
- DAX (Data Analysis Expressions)
